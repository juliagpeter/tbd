Critério de Categorização

Os 80 termos mais presentes na base de dados foram categorizados seguindo a estrutura tecnológica moderna de desenvolvimento de software:

  • Linguagens de Programação (20 termos): Linguagens de alto e baixo nível utilizadas para desenvolvimento
  • Frameworks e Bibliotecas (15 termos): Ferramentas que aceleram o desenvolvimento de aplicações
  • Bancos de Dados (10 termos): Sistemas de gerenciamento de dados relacionais e não-relacionais
  • Cloud e DevOps (10 termos): Tecnologias de computação em nuvem e automação de processos
  • Metodologias e Práticas (8 termos): Metodologias de desenvolvimento e boas práticas
  • Sistemas Operacionais (5 termos): Plataformas de execução de aplicações
  • Tecnologias Web (7 termos): Tecnologias específicas para desenvolvimento web
  • Mobile (3 termos): Plataformas e tecnologias para desenvolvimento móvel
  • Inteligência Artificial (2 termos): Tecnologias emergentes de IA e ML
  • Outros: Termos que não se encaixam nas categorias principais

Metodologia

A análise foi realizada selecionando os 80 termos mais frequentes na base de dados e agrupando-os em categorias baseadas na taxonomia tecnológica atual. Para cada categoria e ano, foi calculada a participação média dos termos pertencentes àquela categoria, proporcionando uma visão agregada da evolução de cada segmento tecnológico ao longo do tempo.
